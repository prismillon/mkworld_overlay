Setup Instructions

English Instructions

How to use this overlay in OBS:

  1. Enter your player name above and click "Get MMR"
  2. Click "Copy Overlay URL" button
  3. In OBS, add a Browser Source
  4. Paste the URL in the URL field
  5. Set Width: 400, Height: 100
  6. Check "Shutdown source when not visible"
  7. Check "Refresh browser when scene becomes active"

日本語の説明

OBSでこのオーバーレイを使用する方法：

  1. 上にプレイヤー名を入力して「Get MMR」をクリック
  2. 「Copy Overlay URL」ボタンをクリック
  3. OBSでブラウザソースを追加
  4. 完全なURLをURLフィールドにペースト
  5. 幅: 400、高さ: 100に設定
  6. 「表示されていない時にソースをシャットダウン」をチェック
  7. 「シーンがアクティブになった時にブラウザを更新」をチェック
